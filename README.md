It is a project with the collection of words from Frankenstein. They are less popular and can help widen vocabulary for B2-C1 learners.

It has 2 features to learn and train words:
- Compose feature that offers you to write a text with the randomly shuffled words
- Flashcard learning with 15 or 30 second timer

To start the project, download the files, go to Terminal, npm start and go to localhost:3000.

If you want to add new words, I recommend adding new words in batches using any AI to pull the definition and etymology and that can connect to the simple database.

In the future I plan to properly host it and let people sending their composed pieces (up to 600 words) for moderation, and if they are written in the spirit of the project they will be published on the separate page.
