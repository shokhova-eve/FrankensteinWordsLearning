About the Project

Notes from the author's desk

This web application is meant to accompany you in reading — or rereading — Frankenstein by Mary Shelley.

Despite being at an advanced level of English, I am still far from the real freedom that native speakers experience. I feel limited in my vocabulary when I write fiction or read certain classic literature, and I wanted to change that.

Recalling words through the flashcard method, searching for them, and using them to write my own texts are, for me, the best ways of doing so.

If you'd like to build a similar project yourself — with the help of an LLM, or without — you can find the source code on my GitHub.
